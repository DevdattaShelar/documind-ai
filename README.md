# DocuMind AI

A multilingual Retrieval-Augmented Generation (RAG) system that answers questions from documents with citations, detects contradictions between documents, and avoids hallucinations.

## Features

- FastAPI backend
- Streamlit UI
- ChromaDB vector database
- Local BGE embeddings
- Groq LLM
- Multilingual support
- Answer citations
- Contradiction detection
